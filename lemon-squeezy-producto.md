# Finzen — Plantilla de App de Finanzas Personales (React + Vite)

## Nombre del producto
Finzen — Personal Finance Dashboard Template

## Tagline (corto, para listado)
Plantilla completa de finanzas personales en React: dashboard, deudas, patrimonio y análisis listos para producción.

## Descripción larga (para la página de producto)

**Finzen** es una plantilla profesional de aplicación web para gestión de finanzas personales, construida con React + Vite y lista para desplegar en minutos. Ideal para desarrolladores que quieren lanzar un SaaS de finanzas, freelancers que necesitan una base sólida para un cliente, o cualquiera que quiera tener control total de su dinero con una app propia.

Incluye todo el código fuente, sin dependencias ocultas ni licencias por uso. Es 100% tuyo: personalízalo, rebrandéalo y publícalo donde quieras (web, PWA instalable, o como base para apps móviles).

### ¿Qué incluye?

- **Dashboard general**: saldo total, ingresos y gastos del mes, tasa de ahorro, alertas inteligentes (pagos próximos, carga de deuda alta) y gráfica de tendencia de 1M/3M/6M.
- **Cuentas**: control de múltiples cuentas (efectivo, débito, crédito, inversiones).
- **Patrimonio**: vista consolidada de activos y pasivos a lo largo del tiempo.
- **Deudas**: seguimiento de tarjetas y préstamos, con alertas de vencimiento y porcentaje de carga sobre ingresos.
- **Análisis**: gráficas de ingresos, gastos y ahorro neto, categorización automática.
- **Historial**: listado completo de transacciones con filtros.
- **Datos**: importación/exportación de información.
- **Modo oscuro** y diseño responsive (mobile-first).
- **PWA instalable**: los usuarios pueden "instalar" la app en su celular/escritorio y usarla sin conexión.

### Stack técnico

- React 18 + Vite
- TailwindCSS
- Recharts (gráficas)
- Arquitectura de componentes limpia y fácil de extender
- Listo para desplegar en Vercel, Netlify o cualquier hosting estático

### Por qué comprar esta plantilla

- Ahorra semanas de desarrollo de UI, gráficas y lógica de cálculo financiero.
- Código limpio, comentado y organizado por páginas/componentes.
- Base perfecta para construir tu propio SaaS de finanzas o app de presupuesto.
- Fácil de conectar a tu propio backend (Supabase, Firebase, API REST, etc.).
- Personalización en un solo archivo: edita `src/config.js` para cambiar el nombre de la app, el ícono, la moneda y el idioma — los cambios se aplican automáticamente en el sidebar, el título y la app instalable (PWA).
